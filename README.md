# PhantomView v0.2.5

A live chat extension for Solana memecoin traders. Chat with others viewing the same token on BullX or Axiom.

## 🚀 New in v0.2.5

### ✨ Major Features
- **Channel-Based Chat System**: Each token now has its own dedicated chat channel
- **Multi-Screen Setup**: Streamlined 3-step process (Username → Website → Token)
- **User Management**: Automatic user joining/leaving with active users list
- **Social Media Integration**: Quick access to our social channels
- **Manual Overlay Control**: Overlay only opens when you explicitly join a channel

### 🔧 Technical Improvements
- **Chrome Storage**: Efficient local storage for channels and messages
- **Automatic Cleanup**: Empty channels are automatically deleted
- **Real-time Sync**: Messages appear instantly across all instances
- **Enhanced Error Handling**: Better connection and user feedback

## 📦 Installation

1. Download the latest release from GitHub
2. Extract the ZIP file
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" in the top right
5. Click "Load unpacked" and select the extracted folder
6. The PhantomView extension will appear in your extensions list

## 🎯 How to Use

### 1. Setup Your Username
- Click the PhantomView extension icon
- Enter your desired username
- Click "Continue"

### 2. Select Website
- Choose between BullX or Axiom
- Click your preferred platform

### 3. Join a Channel
- Enter the coin token you want to chat about
- Click "Join Channel"
- The overlay will open with the specific token's chat

### 4. Chat Features
- **Messages**: Type and send messages (appear as "Me: [message]")
- **Active Users**: See who else is in the channel
- **Minimize**: Click the minimize button to hide the overlay
- **Close**: Click the X button to leave the channel and close overlay
